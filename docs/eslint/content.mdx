## Rules
### `class-order`
Enforce a consistent and logical order of classes.
```json name=.eslintrc.json
{
    "rules": {
        "@master/css/class-order": "warn"
    }
}
```

### `class-validation`
Detect syntax errors early when writing classes.
```json name=.eslintrc.json
{
    "rules": {
        "@master/css/class-validation": "error"
    }
}
```
Set `disallowUnknowClass: true` to disallow using traditional classes:
```json name=.eslintrc.json
{
    "rules": {
        "@master/css/class-validation": ["error", {
            "disallowUnknowClass": true
        }]
    }
}
```
