By default, Master CSS uses token values to [simplify native CSS property values](/docs/rules#default-reference).

In addition, you can customize [global values](/docs/values) and [rules](/docs/rules) to implement your design system, for example:
```js name=master.css.js
export default {
    values: {
        sm: 16,
        md: 32,
        …
    }
}
```
Apply them:
```html
<div class="width:**sm** font:**md**">
```
