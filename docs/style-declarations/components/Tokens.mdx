By default, Master CSS uses predefined variables to simplify native values. You can customize [variables](/docs/variables) to implement your design system, for example:
```js name=master.css.js
export default {
    variables: {
        width: {
            sm: 16,
            md: 32,
            …
        }
    }
}
```
Apply them:
```html
<div class="width:**sm** font:**md**">
```
