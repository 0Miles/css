## Basic usage
### Set the cursor color
Use the `caret-color:color` to change the cursor color in text input.
<InteractingIndicator>Focus on input to see the cursor color</InteractingIndicator>

<Demo class="py:0">
    <input type="text" className="p:5|10 caret-color:pink-70 ~caret-color|.3s outline:none bg:white b:2px|solid|gray-83 b:2px|solid|pink-70:focus r:8" />
</Demo>
```html
<input type="text" class="**caret-color:pink-70**">
```

---
