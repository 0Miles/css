## Getting Started

If you don't have one, you can interactively create a new Nuxt.js project by running:
```bash
npx nuxi init <project-name>
```
```bash
cd <project-name>
```
```bash
npm i @master/css-extractor.vite@beta -D
```

Add `CSSExtractorPlugin` in `nuxt.config.js`:
```js
import { CSSExtractorPlugin } from '@master/css-extractor.vite'

export default defineNuxtConfig({
    vite: {
        plugins: [
            CSSExtractorPlugin()
        ]
    }
})
```
Import the virtual CSS module `master.css` into the entry file `app.vue`:
```html
<script>
    import 'master.css'
</script>
```

Now start the Nuxt.js development server:
```bash
npm run dev -- -o
```
